# How To Run:
Make sure to have .env file on ./express/ containing this:
```
JWT_SECRET_KEY = "67" # Anything you want
TOKEN_HEADER_KEY = "Authorization" # The header request for JWT must be this!
```

React
```
yarn
yarn run dev
```
Express
```
cd express
yarn
yarn run dev
```
